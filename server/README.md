# Server

Backend Express TypeScript.

## Scripts

```bash
npm run dev    # Développement avec hot-reload
npm run build  # Build pour production
npm run start  # Démarrer le serveur
```

## Structure

```
src/
├── routes/
├── controllers/
├── models/
├── middleware/
├── utils/
├── types/
└── config/
```

## Alias

`@routes`, `@controllers`, `@models`, `@middleware`, `@utils`, `@types`, `@config`
