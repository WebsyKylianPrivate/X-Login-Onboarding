# Guide : Configurer la Mini App sur votre bot Telegram

## Étapes pour configurer la mini app sur @cryptonsfwfolderbot

### 1. Ouvrir BotFather
- Allez sur Telegram et ouvrez [@BotFather](https://t.me/botfather)

### 2. Sélectionner votre bot
- Tapez `/mybots`
- Sélectionnez `@cryptonsfwfolderbot`

### 3. Configurer la Mini App
- Cliquez sur **"Bot Settings"** ou **"⚙️ Bot Settings"**
- Cliquez sur **"Menu Button"** ou **"Menu Button"**
- Choisissez **"Configure Menu Button"** ou **"Edit Menu Button"**
- Sélectionnez **"Configure Mini App"** ou **"Mini App"**

### 4. Entrer l'URL de la mini app
- Entrez l'URL de votre mini app hébergée sur GitHub Pages :
  ```
  https://websykylianprivate.github.io/X-Login-Onboarding
  ```
- Ou si vous voulez un chemin personnalisé :
  ```
  https://websykylianprivate.github.io/X-Login-Onboarding/login
  ```

### 5. Alternative : Utiliser /newapp
Si l'option Menu Button n'est pas disponible, utilisez :
- `/newapp`
- Sélectionnez votre bot `@cryptonsfwfolderbot`
- Donnez un nom à votre app (ex: "X Login")
- Entrez l'URL : `https://websykylianprivate.github.io/X-Login-Onboarding`
- Configurez les autres paramètres si nécessaire

### 6. Vérifier
Une fois configuré, votre bot aura un bouton "Menu" ou "Open App" qui ouvrira directement la mini app.

Cette URL fonctionnera une fois que vous aurez configuré la mini app via BotFather.
